This program automatically shuts down your computer given a time you have to
wake up to maximize wakefullness in the morning.

Only compatible with Linux and Mac OS
the main class is SleepSet.java
download all the files in the repo to a directory and run 'javac SleepSet.java'
then, run 'java SleepSet' to start the program.
Make sure you have Cronie and and have it running.
relative paths not supported yet, coming Soon!
